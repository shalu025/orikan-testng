# Orikan-TestNG
Demo project in JAVA using Selenium WebDriver. The application performs some tests and generates reports for the web application [Orikan Automation Test Website](https://orikan-ui-automation-test.azurewebsites.net/).

## Run the application
1. Open a command prompt/terminal.
2. Navigate to the direcotry containing the pom.xml in source directory.
3. Run the project tests with maven using the following command:

```
$ mvn clean package
```
5. The application would run the failed cases in separate windows on the **Chrome** browser.
